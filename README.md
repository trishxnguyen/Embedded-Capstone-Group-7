# Embedded-Capstone-Group-7

## Build Instructions

### You must build using "Raspberry Pi Pico" VS Code Extension
### - From Visual Studio Code, choose "Open Folder" and input the path RTOS_CLASSROOM file path
### - From the "Raspberry Pi Pico Project" side bar, created upon installation of the extension, select "Import Project"
### - Input the path to the RTOS_CLASSROOM directory
### - Select "Compile Project" from the "Raspberry Pi Pico" side bar menu option
### - Put Pico in Boot mode (hold the BOOTSEL button while plugging the Pico into the computer via USB)
### - Once the new drive is selected select "Run Project" (this will compile the project then flash to the Pico)
### - If you need to manually flash the Pico, drag and drop the .uf2 file generated in the "build" directory into the extended storage device of the Pico
### - The storage device will eject then the Pico will reset.
